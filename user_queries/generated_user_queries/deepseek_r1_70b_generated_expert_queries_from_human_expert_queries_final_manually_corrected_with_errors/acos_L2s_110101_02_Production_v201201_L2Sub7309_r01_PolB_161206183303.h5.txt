A script that performs the following steps:

1. Define a constant FILE_NAME with the name of the HDF5 file to be processed.

2. Open the specified HDF5 file in read-only mode.

3. Extract datasets from the file:
   - `xco2` for xCO2 data
   - `sounding_latitude_geoid` for latitude data
   - `sounding_longitude_geoid` for longitude data
   - `sounding_altitude` for altitude data
   - `sounding_time_tai93` for time data

4. Retrieve the 'Unit' attribute from both `xco2` and altitude datasets at index 0.

5. Create a figure with two subplots:
   - First subplot: Orthographic projection map with low resolution, centered at lat_0 = -55 and lon_0 = 120, 
   displaying the sounding trajectory using latitude and longitude coordinates.
   - Second subplot: Line plot showing time vs xco2 data, with a twin axis for time vs altitude data.

6. Customize the map by adding:
   - Coastlines with a line width of 0.5
   - Parallels at latitudes [-80, -60, -40, -20, 0]
   - Meridians at longitudes from -180 to 180 with interval of 20

7. Customize both plots by setting appropriate axis labels, titles, and tick formats.
