A Python script that processes an HDF5 file containing geospatial data. The script should follow these steps:

1. Define Constants: Set a constant `FILE_NAME` to the path "ATL11_051911_0321_006_06.h5".

2. Open and Read HDF5 File: 
Open the file in read mode and access the datasets `latitude`, `longitude`, and `h_corr` from `pt1`. 
Extract attributes `Units` and `long name` using decoding with 'ascii' and 'replace', and retrieve `_Fillvalue` from the h_corr dataset.

3. Data Preprocessing:
   - Convert h_corr data to float.
   - Replace `_Fillvalue` with numpy NaN.
   - Mask data where values are NaN.

4. Create Map Projection: Use an equidistant cylindrical projection with low resolution, setting latitude from -90 to 90 and longitude from -180 to 180.

5. Customize Map:
   - Add coastlines with a line width of 0.5.
   - Draw parallels at intervals of 30 degrees starting from -90 till 120.
   - Draw meridians every 45 degrees from -180 to 180.

6. Generate Scatter Plot: Create a scatter plot using longitude and latitude coordinates, 
   with data as the color variable. Use a marker size of 0.1, jet colormap, no edge colors, and no line widths.

7. Add Colorbar and Title:
   - Attach a colorbar below the map, labeled with the dataset's units.
   - Set the title to include the file basename, dataset path, and long_name from attributes.
