A Python script that processes an HDF5 file and visualizes data on a map. The script should:

1. Define the constant FILE_NAME as 'ATL09_20181117090604_07600101_006_02.h5'.

2. Open the HDF5 file in read mode and access three datasets:
   - 'latitude' for latitude data.
   - 'longitude' for longitude data.
   - 'apparent_surf_reflec' as the main data.
   all from the sub group path `/profile_1/high_rate`
   
3. Read attributes 'units' and 'long_name' using decode('ascii', 'replace') from the apparent_surf_reflec dataset.

4. Create a map with an equidistant cylindrical projection, low resolution, latitude from -90 to 90, longitude from -180 to 180.

5. Add coastlines with linewidth 0.5, parallels at [-90, -60, -30, 0, 30, 60, 90, 120] and meridians at [-180, -135, -90, -45, 0, 45, 90, 135, 180].

6. Generate a scatter plot using longitude and latitude data, with markers of size 0.1, colormap 'jet', no edge colors, and linewidth 0.

7. Add a colorbar labeled with the main dataset's units and set the title to include the file basename and long_name.
