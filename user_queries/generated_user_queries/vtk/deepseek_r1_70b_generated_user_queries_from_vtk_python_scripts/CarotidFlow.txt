    - Apply an outline filter (`vtkOutlineFilter`) to the reader's output.
    - Create an actor (`outlineActor`) for the outline and set its color to black.

12. Assemble the Scene:
    - Add all actors (`outlineActor`, `streamerActor`, `isoActor`) to the renderer.
