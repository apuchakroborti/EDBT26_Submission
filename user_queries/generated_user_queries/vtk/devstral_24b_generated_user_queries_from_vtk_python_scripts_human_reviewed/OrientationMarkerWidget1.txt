A Python script is designed to create an interactive 3D visualization using VTK (Visualization Toolkit) modules, with a focus on displaying an orientation marker widget alongside a superquadric shape in a rendering window.

### Overall Purpose and Functionality
The primary purpose of this script is to set up a VTK-based 3D visualization environment. It reads a polydata file representing an icon, sets up a renderer with specific background color, creates an orientation marker widget, generates a superquadric shape, and displays everything in an interactive window.
