 HDF5 Dataset Access Sub-query:
```python
import h5py

# Open the HDF5 file
with h5py.File('TES-Aura_L2-O3-Nadir_r0000002433_F08_12.he5', 'r') as f:
    # Extract ozone data from '/HDFEOS/SWATHS/O3NadirSwath/Data Fields/O3' (all rows, 6th column)
    ozone = f['/HDFEOS/SWATHS/O3NadirSwath/Data Fields/O3'][:,5]
    
    # Retrieve latitude and longitude from their respective paths
    lat = f['/HDFEOS/SWATHS/O3NadirSwath/Geolocation Fields/Latitude'][:]
    lon = f['/HDFEOS/SWATHS/O3NadirSwath/Geolocation Fields/Longitude'][:]
```
NumPy Data Preprocessing Sub-query:
```python
# Extract Units, Title, and FillValue attributes from the O3 dataset
units = f['/HDFEOS/SWATHS/O3NadirSwath/Data Fields/O3'].attrs['Units']
title = f['/HDFEOS/SWATHS/O3NadirSwath/Data Fields/O3'].attrs['long_name']
fillvalue = f['/HDFEOS/SWATHS/O3NadirSwath/Data Fields/O3'].attrs['_FillValue']

# Replace FillValues with NaNs in the ozone data
ozone[ozone == fillvalue] = np.nan
```
Plotting and Visualization Sub-query:
```python
import matplotlib.pyplot as plt
from mpl_toolkits.basemap import Basemap, cm

# Create a map with cylindrical projection, low resolution, lat limits [-90, 90], lon limits [-180, 180]
map = Basemap(projection='cyl', llcrnrlat=-90, urcrnrlat=90, llcrnrlon=-180, urcrnrlon=180, resolution='l')

# Draw coastlines (linewidth=0.5), parallels (every 45 degrees) from -90 to 90, and meridians (every 45 degrees with labels on left and right) from -180 to 180
map.drawcoastlines(linewidth=0.5)
map.drawparallels(np.arange(-90., 90., 45.), labels=[True, False, False, True])
map.drawmeridians(np.arange(-180., 180., 45.), labels=[True, False, False, True], fontsize=12)

# Scatter plot ozone data using longitude, latitude coordinates; set marker size to 1
x, y = map(lon, lat)
map.scatter(x, y, s=1)

# Add colorbar labeled with O3 Units
cbar = plt.colorbar(cm.ScalarMappable(norm=None, cmap='jet'), shrink=0.8, pad=0.05)
cbar.set_label('%s' % units)

# Set title combining HDF5 filename base and O3 long_name attribute
plt.title('%s - %s' % (f.filename.split('/')[-1], title))
```
