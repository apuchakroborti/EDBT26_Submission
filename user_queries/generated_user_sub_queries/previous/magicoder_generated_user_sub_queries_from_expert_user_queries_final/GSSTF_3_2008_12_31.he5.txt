NumPy Data Preprocessing Sub-query:
```python
import numpy as np

# Replace _FillValue with np.nan and create a masked array
data = np.array(dataset)
masked_data = np.ma.masked_where(np.isnan(data), data)
```

Plotting and Visualization Sub-query:
```python
import matplotlib.pyplot as plt
from mpl_toolkits.basemap import Basemap

# Construct a map with cylindrical projection
map = Basemap(projection='cyl', llcrnrlat=-90, urcrnrlat=90, llcrnrlon=-180, urcrnrlon=180)

# Add parallels and meridians
parallels = np.arange(-90., 90, 30.)
meridians = np.arange(-180., 180., 45.)
map.drawparallels(parallels, labels=[1,0,0,0], fontsize=10)
map.drawmeridians(meridians, labels=[0,0,0,1], fontsize=10)

# Visualize the data using pcolormesh
longitude = np.arange(len(masked_data[0]))
latitude = np.arange(len(masked_data))
map.pcolormesh(longitude, latitude, masked_data, latlon=True)

# Add a colorbar and a descriptive title to the plot
plt.colorbar()
plt.title('Geospatial Data Visualization')
```
